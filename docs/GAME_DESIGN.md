# TheSevenAutoBattle — 게임 디자인 문서

> 최종 업데이트: 2026-04-28

---

## 1. 게임 개요

- **장르:** 덱빌딩 로그라이크 + 오토배틀 (PvE)
- **플랫폼:** Steam (PC)
- **핵심 한 줄:** "정보는 다 줄게. 그걸로 얼마나 아끼냐가 실력이야."
- **수익모델:** 스팀 단일 구매 (인앱결제 없음)

---

## 2. 핵심 루프

```
[런 시작] 빈 로스터로 시작

[매 라운드]
  1. 재화 지급
  2. 적 편성 완전 공개
  3. 상점: 카드 4장 랜덤 공개
     └→ 구매 → 내 로스터에 영구 추가 (런 내)
     └→ 리롤 → 재화 소모, 새 4장 공개
  4. 내 로스터에서 출전 카드 선택 (코스트 한도 내)
  5. 오토배틀
  6. 클리어 → 다음 라운드

[런 종료] 패배 or 보스 클리어 → 로스터 초기화
```

**핵심 긴장감:**
- 최소 재화로 클리어할수록 카드 구매/업그레이드 여유가 생김
- 적 편성이 완전 공개되므로 카운터 조합 선택이 핵심 의사결정
- 코스트 한도 업그레이드 vs 카드 구매 사이의 자원 딜레마

---

## 3. 카드(유닛) 시스템

### 기본 규칙
- 카드 구매 → 내 로스터에 영구 추가 (런 내 유지)
- 같은 카드 여러 장 구매 가능 → 여러 명 소환
- 매 라운드 로스터에서 출전 카드를 직접 선택
- 출전 가능 인원: **총 코스트 한도 이내** (한도는 업그레이드 가능)

### 에셋 (Tiny RPG Character Asset Pack v1.03)
아군 유닛 9종, 적 몬스터 11종 사용.

---

## 4. 시너지 시스템 — 2차원 구조

각 유닛은 **[죄종] 1개 + [직업] 1개**를 보유.
두 축의 시너지가 동시에 적용된다.

### 4-1. 죄종 (7종) — 특성 시너지

| 죄종 | 시너지 효과 방향 |
|------|-----------------|
| **분노** | 전투 중 킬할수록 공격력 누적 상승 |
| **탐욕** | 전투 후 재화 추가 획득 or 코스트 절감 |
| **교만** | 아군 수가 적을수록 개인 스탯 폭등 |
| **나태** | 적에게 지속 디버프 (독, 약화, 슬로우) |
| **색욕** | 적 유닛 일시 매혹 (아군처럼 싸움) |
| **폭식** | 처치한 적의 스탯 일부 흡수 |
| **질투** | 가장 강한 적의 버프 복사 |

### 4-2. 직업 (아군 유닛 9종) — 역할 시너지

| 직업 | 유닛 | 역할 |
|------|------|------|
| **Soldier** | Soldier | 저코스트 기본 보병 |
| **Axeman** | Armored Axeman | 돌격형 전사 |
| **Swordsman** | Swordsman | 균형형 검사 |
| **Knight** | Knight | 중장 탱커 |
| **Templar** | Knight Templar | 중장 창기사 |
| **Lancer** | Lancer | 기마 돌격 |
| **Archer** | Archer | 원거리 딜러 |
| **Priest** | Priest | 힐/버프 지원 |
| **Wizard** | Wizard | 광역 마법 딜러 |

### 4-3. 2차원 시너지 예시

```
[Knight][교만] — 혼자 남을수록 스탯이 폭발적으로 오르는 탱커
[Archer][질투] — 가장 강한 적의 버프를 복사하는 원거리 딜러
[Priest][탐욕] — 힐할 때마다 재화를 추가 획득하는 지원가
[Wizard][나태] — 광역 공격에 슬로우+독을 동시에 거는 마법사
```

## 5. 업그레이드 시스템 — 분기 스킬트리

카드마다 고정된 단일 업그레이드가 아닌, **분기형 스킬트리** 방식.
같은 카드도 런마다 다른 방향으로 성장 가능.

### 구조 예시 — Soldier

```
[Soldier] 기본 (체력 100 / 공격 10 / 코스트 2)
        │
        ├─ [A] 돌격형 — 체력 +20, 공격 +15
        │         ├─ [A-1] 베서커: 킬 시 공격력 +5 (스택)
        │         └─ [A-2] 듀얼블레이드: 공격 2회 히트
        │
        └─ [B] 방어형 — 체력 +40, 방어막 추가
                  ├─ [B-1] 가디언: 피격 시 인접 아군 방어막 부여
                  └─ [B-2] 카운터: 피격 시 공격자에게 반격
```

### 업그레이드 × 죄종 시너지

```
[Soldier][분노] + 베서커 업그레이드
→ 킬할수록 분노 스택 + 베서커 스택 동시 적용 → 후반 폭발
```

### 업그레이드 시점 (미결)
- 후보 A: 라운드 클리어 후 재화 소모
- 후보 B: 특정 라운드마다 자동 해금
- 후보 C: 보스 클리어 보상
