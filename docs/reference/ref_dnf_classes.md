## 기본 직업 16종 (전직 포함)

### Slayer (귀검사) — 남/여
근접 검술 기반. **올라운드 근접 딜러**. 공격·방어·속도 균형.

| 전직 | 특징 |
|---|---|
| Blademaster (웨펀마스터) | 다양한 무기 숙련, 범용성 최고 |
| Soul Bender (소울브링어) | 귀신 소환, 디버프+딜 |
| Berserker (버서커) | HP 소모해 공격력 폭증, 극딜 |
| Asura (아수라) | 기(氣) 기반 마법+물리 혼합 |
| Ghostblade (검귀) | 빠른 연속기, 유령 칼날 |
| Swordmaster (여귀검사) | 검기 방출, 원거리+근거리 혼합 |
| Demon Slayer (데몬슬레이어) | 악마 계약, 공격+생존 |
| Vagabond (베가본드) | 두 자루 검, 고기동 연속기 |
| Dark Templar (다크템플러) | 영혼 흡수, 탱킹+딜 |
| Spectre (스펙터) | 저주 기반 독립형 딜러 |

### Knight (나이트) ⭐ 만능형 참고 핵심
방패+검. 공격·방어·유틸리티 모두 보유.

| 전직 | 특징 |
|---|---|
| **Elven Knight (엘븐나이트)** | **만능형의 대표 사례** — 슈퍼아머·방어력 증가·받는 피해 감소·HP 회복·홀딩 스킬을 동시에 보유. "다재다능한 직업"으로 명시됨. 체인 러쉬로 스킬 캔슬+딜 증가, 리듬형 조작 |
| Chaos (카오스) | 소검 기반, 방패고 시스템 활용 |
| Lightbringer (라이트브링어/팔라딘) | 둔기+방패, 성스러운 딜러 |
| Dragon Knight (드래곤나이트) | 도(刀)+드래곤 방패, 고화력 |
