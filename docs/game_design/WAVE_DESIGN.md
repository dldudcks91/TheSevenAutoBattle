# 웨이브 디자인

> 한 런 = 10웨이브. 각 웨이브가 어떤 학습 포인트를 시험하는지의 의도를 담는다.
> 적 라인업·카운트 등 데이터는 [src/data/rounds.csv](../src/data/rounds.csv) 에서 관리.

---

## 1. 디자인 원칙

1. **웨이브 1개 = 메카닉 1개 학습** — 새 적은 한 웨이브에 1종만 도입하고, 다음 웨이브에서 응용·혼합한다.
2. **챕터 첫 웨이브는 완충** — 적 수를 살짝 줄이고 신규 메카닉을 단독으로 공개해 학습 시간을 준다.
3. **챕터 마지막 웨이브는 종합 시험** — 챕터 내 모든 메카닉을 섞고, 직전 챕터 적도 일부 잔향으로 남긴다.
4. **보스 wave는 "보스 단일 위협 + 보스를 강하게 만드는 호위"** — 호위 자체보다 호위 효과로 보스의 위협이 증폭되는 구도.
5. **카운트 곡선은 단조 증가, 보스 wave만 살짝 줄이고 보스 강함으로 보충.**

---

## 2. 챕터 구조

| 챕터 | 웨이브 | 테마 | 핵심 학습 |
|---|---|---|---|
| 1 | W1~3 | 오크 | 카드 시스템 흐름 → 방어력 개념 → 숫자 압박 |
| 2 | W4~7 | 언데드 | 원거리 위협 → 부활 → 디버프 → 기동(후방 침투) |
| 3 | W8~10 | 야수 + 혼합 | 처치 타이밍(스노우볼/사망 강화) → 종합 압박 → 보스 |

---

## 3. 웨이브별 의도

### Chapter 1 — 오크 (기초 학습)

| 웨이브 | 학습 포인트 | 압박 강도 |
|---|---|---|
| W1 | 카드 추첨 → 셀 배치 → 전투 시작 흐름 익히기. 실패하기 어려운 입문 라운드. | 매우 약함 |
| W2 | **방어력의 의미** — 평타가 1씩 들어가는 경험. 방어무시(마법) 또는 방어감소(액스맨)의 필요성 인지. | 약함 |
| W3 | 장갑 + 숫자 동시 압박. 2라운드치 누적 골드로 그리드를 확장했는지 시험. | 보통 |

### Chapter 2 — 언데드 (메카닉 도입)

| 웨이브 | 학습 포인트 | 압박 강도 |
|---|---|---|
| W4 | **원거리 위협 첫 등장** — 후방도 안전하지 않음. 독반사 메카닉으로 근접 무뇌 진입 견제. | 보통 |
| W5 | **부활 메카닉** — 한 번 죽여도 다시 옴. 단발 처치 이후 후속타·재처치 동선 학습. | 보통 |
| W6 | **공격력 디버프** — 그레이트소드 스켈레톤이 아군 공격력을 깎음. 공격 분산(차밍·도발) 또는 빠른 처치 압박. | 강함 |
| W7 | **기동 위협 + 챕터 종합 시험** — 오크 라이더가 전선 무시하고 후방 침투. 도발·전선 차단의 가치. 챕터2 전 메카닉 혼합. | 강함 |

### Chapter 3 — 야수 + 혼합 (극단 압박과 보스)

| 웨이브 | 학습 포인트 | 압박 강도 |
|---|---|---|
| W8 | **처치 타이밍 메카닉** — 슬라임 동시 처치(사망 시 강화 회피), 늑대인간 킬 분배(스노우볼 차단). 처치 순서·동시성이 결과를 가름. | 강함 |
| W9 | **보스 직전 종합 점검** — 야수·장갑·원거리·스노우볼이 모두 한 자리에. 자원과 편성이 부족하면 W10 도달 불가. | 매우 강함 |
| W10 | **보스 wave** — 웨어베어 보스 + 엘리트 오크(진입 시 전체 강화) + 늑대인간 호위. 엘리트 오크 버프가 보스 데미지를 증폭하므로 호위를 무시하면 보스가 더 위험해짐. | 보스 |

---

## 4. 적 카운트 곡선 (방향성)

- W1~3: 소수에서 시작해 챕터 끝에 살짝 늘어남
- W4~7: 챕터2 시작 시 한 단 늘리고, 종합 시험에서 정점 근처
- W8~9: 압박 정점 — 가장 많은 수
- W10: 보스 wave는 호위 포함 다소 줄임 — 수보다 보스 강함이 핵심

> 정확한 적 수는 [src/data/rounds.csv](../src/data/rounds.csv) 참조. 본 문서는 의도만 담는다.

## 5. 미결 항목

| 항목 | 비고 |
|---|---|
| 웨이브별 적 수치·라인업 미세 조정 | 본 문서는 방향성만. 플레이테스트 1회 후 카운트와 구성 조정 예정. |
| 보스 고유 메카닉 | 현재 웨어베어는 일반 적과 동일 스탯. 보스 전용 스킬(예: 페이즈 전환, 광폭화) 도입 여부 미결. |
| 챕터 클리어 보상 차등 | 챕터 마지막 웨이브(W3, W7, W10) 보상 가중치 부여할지 미결. |
| 다음 웨이브 사전 공개 깊이 | 현재 라운드 시작 시 100% 공개. 다음 웨이브까지 미리 보여줄지(메타 의사결정) 미결. |
| 챕터 간 휴식·상점 라운드 | 챕터 사이에 전투 없이 카드 정리만 하는 라운드를 둘지 미결. |
