# 유닛 디자인

> 유닛 식별·역할 정의. 스킬 상세 → [SKILL_DESIGN.md](SKILL_DESIGN.md) 참조.

---

## 1. 디자인 원칙

1. **Player fantasy first** — 정체성을 먼저 정하고, 그 정체성을 행동으로 옮기는 스킬을 설계한다. 메커닉에서 정체성을 역산하지 않는다.
2. **시너지 없음** — 종족·직업 시너지 시스템 없음. 유닛 스킬은 개별 독립 행동만.

## 4. 셀 배치 규칙

- 셀은 처음 배치 시 하나의 유닛 타입으로 고정된다
- 같은 유닛 타입이 여러 셀에 독립적으로 존재할 수 있다 — 각 셀은 독립된 스킬·아이템 컨텍스트를 가짐
- 셀에 N명 배치 → 전투 시 N개의 개별 유닛으로 출전
- 셀에 장착된 스킬·아이템 카드는 그 셀 안의 모든 병사에게 동일하게 적용
- **셀당 최대 4명** — 셀 내부는 2×2 사분면(좌상/우상/좌하/우하)으로 분할되어 한 칸에 한 병사씩 점유한다. 5번째 영웅 카드 배치는 거부되며, 다른 셀로 분산해야 한다. SHOP에서 본 사분면 위치가 그대로 전투 출전 좌표로 매핑된다.

## 5. 스킬 카드 시스템

- 스킬은 **카드 형태**로 매 라운드 핸드에 섞여 등장 (영웅·강화·스킬·아이템 4종 중 하나).
- 핸드 5장 추첨 흐름·리롤·1회용 규칙은 [GAME_DESIGN.md §4 카드 시스템](GAME_DESIGN.md) 참고.
- 구매한 스킬 카드는 원하는 영웅에게 장착 (장착 UI/타게팅 동선은 미결).
- 같은 종류가 여러 셀에 있어도 셀 단위로 독립 장착 가능.
- 영웅당 스킬 슬롯: **기본 스킬 1개 (구매 시 제공) + 카드 슬롯 2개** = 최대 3개.
- **현재 상태:** SKILL 카드는 더미 — 클릭 시 골드만 차감되며 실제 장착 효과는 미구현.
