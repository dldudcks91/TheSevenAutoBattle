# 유닛 스킬 설계

> 스킬 시스템 구조 정의. 유닛별 스킬 상세는 별도 작성 예정. 유닛 식별·역할 → [UNIT_DESIGN.md](UNIT_DESIGN.md) 참조.

---

## 1. 디자인 원칙

1. **Trigger = 정체성의 행동** — "이 유닛이라면 이 순간에 무언가를 한다"가 자연스럽게 느껴지는 트리거를 고른다. 메커닉에서 트리거를 역산하지 않는다.
2. **PvE 카운터 픽 가시화** — 적 스킬은 "어떤 아군을 쓰면 쉬워지는가"가 플레이어 눈에 보여야 한다.

---

## 2. 스킬 구성 구조

스킬 하나를 완전히 서술하는 6개 축.

| 축 | 정의 |
|---|---|
| **Trigger** | 언제 발동하는가 |
| **Subject** | 누가 이 트리거를 감지하는가 (Self / 아군 전체 / 전 유닛) |
| **Condition** | 트리거 외 추가 조건 (없으면 —) |
| **Effect** | 무엇을 하는가 (키워드 또는 직접 서술) |
| **Target** | 누구에게 적용되는가 |
| **Duration** | 얼마나 지속되는가 |

---

## 3. 트리거 어휘

트리거는 성격에 따라 세 종류로 나뉜다. 실행 모델이 다르므로 구현 시 주의.

| 종류 | 트리거 | 시점 |
|---|---|---|
| **이벤트형** | **On-Deploy** | 배치 직후 1회 |
| 이벤트형 | **On-Attack** | 공격 발생 시 (피해 적용 포함) |
| 이벤트형 | **On-Kill** | 적 처치 직후 |
| 이벤트형 | **On-Ally-Death** | 아군 사망 직후 |
| 이벤트형 | **On-Death** | 자기 사망 직후 (1회) |
| **임계형** | **On-Damaged-Below-X%** | 자기 체력이 임계치 아래로 떨어지는 순간 1회 |
| **지속형** | **Aura** | 매 틱 지속 적용 |
| 지속형 | **Periodic** | 일정 간격마다 반복 |

---

## 4. 키워드 라이브러리

### 기본 키워드

| 키워드 | 정의 |
|---|---|
| **Strike X** | 대상에게 X 피해 |
| **Cleave X** | 범위 내 대상에게 X 피해 |
| **Heal X** | 대상 체력 X 회복 |
| **Lifesteal X%** | 가한 피해의 X%만큼 자신 체력 회복 |
| **Buff ATK +X** | 공격력 X 증가 |
| **Buff Armor +X** | 방어력 X 증가 |
| **Taunt** | 적이 이 유닛을 우선 타겟으로 삼음 |

### 상태이상

| 키워드 | 정의 |
|---|---|
| **Stun** | 행동 불가 |
| **Freeze** | 이동 속도 감소 |
| **Poison** | 매 틱 지속 피해 |
| **Burn** | 피격 시 고정값 추가 피해 |

## 5. 유닛별 기본 스킬

> 기본 스킬은 영웅 구매 시 고정 제공. 수치(간격·확률·배율)는 데이터 파일에서 관리.

| 유닛 | 스킬명 | 트리거 | 설명 |
|---|---|---|---|
| Soldier | 제압 | On-Attack | 공격 시 일정 확률로 대상을 기절시킨다 |
| Armored Axeman | 전쟁의 함성 | Periodic | 주변 적에게 Taunt를 시전한다 |
| Knight | 불굴의 의지 | On-Attack | 공격할 때마다 가한 피해의 일부만큼 체력을 회복한다 |
| Knight Templar | 전투의 오오라 | Aura | 주변 아군의 공격력을 지속적으로 높인다 |
| Hero | 살육의 성장 | On-Kill | 적을 처치할 때마다 공격력이 영구적으로 오른다 |
| Lancer | 충격 돌격 | On-Deploy | 전투 시작 시 달려나가 대상에게 강력한 피해를 입힌다 |
| Archer | 급소 타격 | On-Attack | 공격 시 일정 확률로 배율 피해를 입힌다 |
| Wizard | 화염구 | Periodic | 주기적으로 대상에게 마법 피해를 입힌다 |
| Priest | 치유의 기도 | Periodic | 주기적으로 아군 중 체력이 가장 낮은 유닛을 치유한다 |
